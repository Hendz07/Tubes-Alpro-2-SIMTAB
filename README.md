# Tubes-Alpro-2-SIMTAB
Repository ini berisi source code aplikasi SIMTAB (Sistem Manajemen Tagihan Bulanan) yang dibuat menggunakan Go (Golang). Aplikasi menyediakan fitur pengelolaan tagihan, pencarian data dengan Sequential Search dan Binary Search, pengurutan menggunakan Selection Sort dan Insertion Sort, serta statistik pembayaran tagihan.
